在LCD屏幕上显示一个字符串

如果字符串宽度或者高度为0，就默认字符串以（x，y）和（width，height）为文本框，注width为屏幕宽度，height为屏幕高度

参数
x字符串左上角x坐标
y字符串左上角y坐标
str要显示的字符串
string_width字符串宽度，默认为0
string_height字符串高度，默认为0
size字体大小，默认为DEFAULT_FONT_SIZE
color字体颜色，默认为DEFAULT_PENCIL_COLOR
back_color背景颜色，默认为DEFAULT_BACK_COLOR
